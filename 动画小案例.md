## 动画小案例

```
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>hover动画</title>
	<style>
		div {
			width: 200px;
			height: 200px;
			background: red;
			opacity: 0;
			transition: all 2s linear;
		}
		div:hover {
			transition: all 2s linear;
			transform: translate(50%,0) rotate(360deg);
			width: 300px;
			height: 300px;
			opacity: 1;
		}
	</style>
</head>
<body>
	<div>hover动画：2秒之内旋转360度，放大1.5倍，右移动50%，透明度从0变1，动画类型匀速</div>	
</body>
</html>
```
